Interactive Linear Algebra

Date: 2025   Status: Two demos done

While learning the concepts of linear algebra, I would imagine little animations of the math problems. I realized it might be interesting to make these into actual interactive demos.


These demos work using the mouse cursor, so it would be difficult to use them on mobile
